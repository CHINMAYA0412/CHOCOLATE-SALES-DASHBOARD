# 🍫 Chocolate Sales Dashboard

## 📌 About the Project

This project is an interactive **Power BI Dashboard** created using a Chocolate Sales dataset. It helps analyze sales performance, products, countries, and salespersons through easy-to-understand visualizations.

---

## 🎯 Objective

- Analyze chocolate sales data.
- Find top-selling products.
- Compare sales across countries.
- Track salesperson performance.
- Identify monthly sales trends.

---

## 🛠️ Tools Used

- Power BI
- Excel / CSV
- Power Query
- DAX

---

## 📂 Dataset

The dataset includes:

- Sales Person
- Country
- Product
- Date
- Amount
- Boxes Shipped

---

## 📊 Dashboard Features

- Total Sales
- Total Boxes Shipped
- Sales by Country
- Sales by Product
- Top Salespersons
- Monthly Sales Trend
- Interactive Filters (Country, Product, Sales Person)

---

## 💡 Key Insights

- Identified the highest-selling products.
- Compared sales across countries.
- Found the top-performing salespersons.
- Analyzed monthly sales trends.
- Reviewed shipment distribution.
